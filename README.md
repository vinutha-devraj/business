# Postman Project

## Description
This is a web project containing HTML, CSS styling, and image assets.

## Folder Contents
- `index.html` - Main HTML file with page content
- `styles.css` - CSS stylesheet for styling
- `vinutha.jpg` - Image asset

## How to Use
1. Open `index.html` in your web browser to view the project
2. The styling is applied from `styles.css`
3. Images are displayed from the folder

## File Details
- **index.html** - Contains the main page structure and content
- **styles.css** - Provides styling and layout for the project
- **vinutha.jpg** - Image file used in the project

## Getting Started
Simply open `index.html` in any web browser to view the project.
